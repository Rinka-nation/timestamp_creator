YouTube Timestamp Creator - README

1. 拡張機能の概要

この拡張機能は、YouTubeの動画にタイムスタンプ付きのメモを作成・管理するためのツールです。動画を見ながら気になった瞬間の時間を記録し、コメントや説明文で使える形式のタイムスタンプリストを簡単に作成できます。


2. 主な機能

 タイムスタンプの記録・編集
- タイムスタンプ追加: ショートカットキー（Shift + Enter または P）で、動画の現在再生時間を記録できます。
- テキスト編集: 記録したタイムスタンプに、自由にメモを追記・編集できます。
- 自動保存: 編集内容はリアルタイムで自動的に保存されます。
- 表示/非表示: ショートカットキー（G）で、タイムスタンプ編集ウィンドウの表示・非表示を切り替えられます。

 ポップアップからの各種設定
拡張機能のアイコンをクリックして表示されるポップアップから、以下の設定が可能です。

- タイムスタンプ管理: 保存されている全ての動画のタイムスタンプデータを一覧で確認し、不要なものを削除できます。
- タイムスタンプ形式設定: タイムスタンプが挿入される際の接頭辞（プレフィックス）と接尾辞（サフィックス）をカスタマイズできます。
- NGワード設定: 設定した単語が入力された際に、自動で「〇」に置き換えることができます。これにより、意図しない単語の使用を防ぎます。
- 拡張機能の再読み込み: ポップアップ右上の更新ボタンから、拡張機能の表示を再読み込みできます。

 シャドウバン警告
YouTubeのコメントでシャドウバンの原因となる可能性がある特定のタイムスタンプ（例: `1:10`）を検知すると、該当箇所がハイライトされ警告が表示されます。


 3. 使い方

1.  YouTubeの動画ページを開くと、画面右側にタイムスタンプ編集用のウィンドウが表示されます。
2.  ショートカットキーを使ってタイムスタンプを追加したり、ウィンドウを直接編集したりします。
3.  編集した内容は自動でブラウザに保存され、次回同じ動画を開いた際に復元されます。
4.  詳細な設定や過去のタイムスタンプの管理は、ブラウザのツールバーにある拡張機能アイコンから行います。

 ショートカットキー一覧
- `Shift + Enter` または `P`: 現在の再生時間のタイムスタンプを追加します。
- `G`: タイムスタンプ編集ウィンドウの表示/非表示を切り替えます。
- `U`: 現在の再生時間をクリップボードにコピーします。
- `Y`: `U`キーでコピーした時間をタイムスタンプとして追加します。
- （表示モード中）タイムスタンプをクリック: 動画をその時間にシークします。
- （表示モード中）タイムスタンプをクリックで選択して `↑` `↓` キー: タイムスタンプの時間を1秒ずつ調整します。
- （表示モード中）ウィンドウをダブルクリック: 編集モードに切り替えます。


 4. 注意点

- ストレージの自動削除: ブラウザの保存容量には上限があるため、容量が一定以上（80%）に達すると、最も古く（最終アクセス日時が古い）保存された動画のタイムスタンプデータが自動的に削除されます。重要なデータは別途バックアップを取ることをお勧めします。
